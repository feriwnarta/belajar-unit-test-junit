Belajar unit test menggunakan framework dari junit

1. membuat unit test baru dengan menggunakan anotation test *
2. menggunakan method assertion *
3. menggunakan method assertion (throws) dengan keinginan hasil dari unit test haru menghasilkan exception *
4. menggunakan display name, untuk mengganti deskripsi file didalam hasil console junit
5. menggunakan display name generator
6. menggunakan assumption, jika keinginan tidak tercapai makan akan melempar error
7. menggunakan beforeeach, aftereach
8. menggunakan disable
9. menggunakan testAbortedException, tes khusus junit untuk menghentikan unit test jika terjadi exception
10. menggunakan tes kondisi lingkup system (properti, variable ,environment, dll) dengan anotation yg sudah ada
11. menggunakan tag dan tags untuk memberi tanda unit test
    !! metod antar unit test indenpenden tidak hidup diobjek yang sama, setiap method unit test dijalankan
    !! setiap method akan menggunakan objek baru
